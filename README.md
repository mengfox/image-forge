# image-forge

用于维护 `zjmf`、`zjmf-finance` 与 `lskypro-v2` 镜像构建配置、示例编排文件和发布 workflow。

## 目录

- `images/zjmf`: ZJMF 业务管理系统运行镜像
- `images/zjmf-finance`: ZJMF 财务系统运行镜像
- `images/lskypro-v2`: Lsky Pro v2 运行镜像
- `.github/workflows`: GitHub Actions 构建与发布流程
- `CHANGELOG.md`: 仓库变更记录

## 使用

各镜像目录下都提供或维护对应的构建资源。
具体构建和运行方式请查看对应目录内的说明文档或 Dockerfile。
